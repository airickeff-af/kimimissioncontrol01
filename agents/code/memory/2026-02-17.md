### Deliverables

#### 1. API Server (`/mission-control/dashboard/api/server.js`)
- RESTful API on port 3001
- CORS-enabled for frontend integration
- Endpoints for agents, tasks, system status, logs, files

#### 2. API Endpoints
| Endpoint | Description |
|----------|-------------|
| `GET /api/agents` | List all agents with status |
| `GET /api/agents/:id` | Get specific agent details |
| `GET /api/agents/:id/activity` | Get agent activity history |
| `GET /api/agents/:id/files` | List agent output files |
| `GET /api/system/status` | System health and metrics |
| `GET /api/system/logs` | Recent system logs |
| `GET /api/system/activity` | Global activity feed |
| `GET /api/tasks` | Task queue (pending/completed) |
| `GET /api/files/browse` | Browse file tree |
| `GET /api/health` | Health check |

#### 3. Client Library (`/mission-control/dashboard/api/client.js`)

- Real-time UI updates
