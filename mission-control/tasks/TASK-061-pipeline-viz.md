## Objective
Create a visual pipeline (Kanban-style) for DealFlow showing deal stages.

## Requirements
1. **Pipeline Stages** (default columns)
   - New Lead
   - Contacted
   - Meeting Scheduled
   - Proposal Sent
   - Negotiation
   - Closed Won
   - Closed Lost

2. **Visual Features**
   - Drag-and-drop deal cards between stages
   - Deal cards show: Company, Contact, Value, Priority
   - Color coding by priority/urgency
   - Stage counters
   - Filter by owner, date, value

3. **Interactive Elements**
   - Click card to view/edit deal details
   - Quick actions (schedule meeting, send email)
   - Add new deal button
   - Search/filter bar

4. **Responsive Design**
   - Desktop: Full Kanban board
   - Mobile: Swipeable columns or list view

## Tech Stack
- Frontend: React/Vue component
- Styling: Tailwind CSS or similar
- State: Existing DealFlow state management

## Deliverables
- Kanban board component
- Deal card component
- Stage management logic
- Integration with existing DealFlow data
